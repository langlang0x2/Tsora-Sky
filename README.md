# Tsora Sky

我的个人博客，基于 [Astro](https://astro.build) 构建，使用 [astro-theme-pure](https://github.com/cworld1/astro-theme-pure) 主题，部署在 [Vercel](https://vercel.com)。

🔗 [tsora.me](https://tsora.me)

## 技术栈

- **框架**: Astro
- **主题**: astro-theme-pure
- **样式**: UnoCSS
- **评论**: Waline + Neon DB
- **部署**: Vercel
- **代码高亮**: Shiki (自定义 transformer)
- **数学公式**: KaTeX

## 本地运行

```bash
bun install
bun dev
```
